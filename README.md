# 🤖 AI Proposal Generator

An AI-powered proposal generation system designed for freelancers and agencies to create high-converting client proposals in seconds.

Instead of starting from a blank page, users answer a few structured questions and instantly receive a tailored, client-ready proposal with natural tone and real-world context.

---

## 🚀 Features

- Structured input system (no blank page problem)
- AI-generated proposals in seconds
- Tone selection (Professional / Friendly / Confident)
- Injects real experience into proposals
- Clean Streamlit interface
- Downloadable proposal output

---

## 🧠 How It Works

1. User fills 5 guided inputs:
   - Project description  
   - Client problem  
   - Proposed solution  
   - Timeline  
   - Experience / proof  

2. The system generates a proposal using a custom prompt optimized for real client conversion.

3. The output is ready to use, edit, or download.

---

## 🛠 Tech Stack

- Python  
- Streamlit  
- LangChain  
- OpenAI API
